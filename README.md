# splitwise-a-smart-bill-splliting-system
SplitWise is a smart bill splitting and expense management system that helps users track expenses, split bills among friends, manage payments, and monitor spending efficiently.
## About the Project

This project is developed using **React.js** and provides an interactive user interface for managing personal and shared expenses. Users can add expenses, divide bills among multiple people, and track who owes money.

## Features

- Add and manage expenses
- Split bills between users
- Track individual balances
- Expense history management
- Categorize expenses
- User-friendly dashboard
- Responsive React-based interface
- Simple and efficient expense tracking

## Technologies Used

- React.js
- JavaScript
- HTML5
- CSS3
- React Components
- Node.js
